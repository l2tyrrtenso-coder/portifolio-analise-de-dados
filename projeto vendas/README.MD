 Projeto 01 —  Vendas



Desenvolvimento de um Dashboard Comercial completo utilizando Power BI, a partir de uma base de dados com mais de 5.000 registros contendo inconsistências estruturais e erros de preenchimento.

O projeto envolveu tratamento, modelagem de dados e construção de indicadores estratégicos para análise de desempenho comercial.

---

Objetivo

Transformar uma planilha desestruturada em um modelo analítico confiável capaz de responder rapidamente:

* Qual o faturamento total?
* Qual o ticket médio?
* Quem são os melhores vendedores?
* Quais produtos geram maior receita?
* Como está a evolução das vendas ao longo do tempo?

---

## 🧹 Etapas do Projeto

### 1️⃣ Limpeza e Tratamento de Dados (Power Query)

* Tratamento de campos vazios (substituídos por “Desconhecido”)
* Conversão de valores monetários de texto para decimal
* Substituição de "null" por 0 em campos numéricos
* Tratamento de erros (error → null → 0)
* Remoção de duplicidades
* Padronização de texto (cidade, vendedor e produtos)
* Correção de tipos de dados

---

### 2️⃣ Modelagem de Dados

* Estruturação da base em modelo relacional
* Criação de tabela fato de vendas
* Definição de métricas utilizando DAX
* Organização em formato analítico para melhor performance

---

### 3️⃣ Criação de Medidas DAX

Principais métricas desenvolvidas:

* Faturamento Total
* Total de Pedidos
* Ticket Médio
* Quantidade Vendida
* Análise temporal de crescimento

---

## 📈 Estrutura do Dashboard

O dashboard foi organizado em blocos estratégicos:

🔹 Filtros Interativos

* Ano
* Mês
* Produto
* Vendedor

🔹 Indicadores Principais (KPIs)

* Faturamento Total
* Total de Pedidos
* Ticket Médio
* Quantidade Vendida

🔹 Análises Visuais

* Evolução mensal do faturamento
* Ranking de produtos
* Performance por vendedor
* Distribuição por cidade

---

## 🛠 Ferramentas Utilizadas

* Excel (base de dados)

## 📊 Resultado

O projeto entregou uma visão estratégica e interativa das vendas, permitindo:

* Identificação dos produtos mais lucrativos
* Análise de performance da equipe comercial
* Monitoramento de crescimento mensal
* Tomada de decisão baseada em dados
